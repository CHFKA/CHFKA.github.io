编写 makefile

发表于 2021-06-05 更新于 2021-12-19

介绍

​ 这是在 linux 环境下编写 makefile 的要点和自己编写的例子。

编写 makefile

三要素

  • 目标
  • 依赖
  • 规则命令

写法

第一版

目标：依赖

tab 键 规则命令

app(可执行文件) ： *.c

​ gcc -o app -I include/ *.c

  • 缺点：如果更改其中一个文件是，所有的源码都重新编译

第二版

  • 定义变量 ---ObjFiles=*.o

  • 变量的使用 ---$(ObjFiles) $(变量名)

  • makefile 的隐含规则：默认处理第一个目标

第三版

函数

  • wildcard 可以进行文件匹配
  • patsubst 内容的替换

变量

  • $@ 代表目标
  • $^ 代表全部依赖
  • $< 第一个依赖
  • $? 第一个变换的依赖
  • clean 清理命令
    • @不输出该条规则
    • ’-‘ 报错后继续执行

第四版

  • clean 存在歧义时 使用.PHONY

补充

  • 使用其他名字的 makefile

    • make -f （文件名）

    • 例子

# makefile
