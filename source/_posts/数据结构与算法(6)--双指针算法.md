# 双指针算法

在快排和归并算法中都有运用。

核心思想：把使用暴力双重for循环的O(n^2)的情况优化到O(n)。

```c++
for（i = 0, j = 0; i < n; i++)
{
	while(j<i && check(i,j)) j++;
}
```

给定一个长度为 n 的整数序列，请找出最长的不包含重复的数的连续区间，输出它的长度

------

实现：

```c++
#include<iostream>
using namespace std;
const int N =100010;
int n;
int a[N],s[N];
int main()
{
    cin >> n;
    for(int i = 0; i < n; i++) cin >> n;
    int res = 0;
    
    for(int i = 0, j = 0; i < n; i++)
    {
        s[a[i]]++;
        while(s[a[i]] > 1)
        {
            s[a[j]]--;
            j++;
        }
        res = max(res, i - j + 1);
    }
    cout << res << endl;
    return 0;
}
```
